# SQL Data Analysis Project

## Overview
This project demonstrates my SQL skills through analysis of a retail sales dataset. The objective is to answer business questions using SQL queries and extract meaningful insights from structured data.

## Dataset
The dataset represents retail sales data including:
- Order ID
- Product Category
- Sales
- Quantity
- Order Date
- Customer Region

## Business Questions
1. Total sales by product category  
2. Monthly sales trend  
3. Top 5 best-selling products  
4. Sales contribution by region  
5. Average order value  

## SQL Concepts Used
- SELECT, WHERE
- GROUP BY, ORDER BY
- COUNT, SUM, AVG
- HAVING
- Subqueries

## Files Included
- `dataset.csv`
- `queries.sql`

## Outcome
This project highlights my ability to use SQL for data analysis and answer real-world business questions.
